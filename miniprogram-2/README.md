# 智车通 - 无人车监管可视化小程序

## 项目简介

智车通是一款面向政务监管的无人车管理小程序，主要用于展示无人车运行情况、审批流程以及执法核验功能。该项目是一个演示型 Demo，重点在于数据可视化、简单业务流程和交互体验。

## 技术栈

- **前端框架**：微信小程序原生开发
- **地图组件**：微信小程序 `map` 组件
- **图表组件**：自定义 Canvas 图表组件
- **网络请求**：wx.request
- **扫码功能**：wx.scanCode
- **样式**：WXSS + CSS3

## 功能模块

### 1. 首页（数据大屏）

#### 核心功能

- **数据统计卡片**：展示总车辆数、在线车辆、异常车辆、今日新增
- **地图展示**：使用 `<map>` 组件展示车辆位置分布
- **图表展示**：
  - 车辆运行趋势（折线图）
  - 报警趋势（柱状图）

#### 交互功能

- 点击地图标记跳转到车辆详情页
- 下拉刷新数据
- 实时显示统计数据

### 2. 车辆详情页

#### 信息展示

- 车辆基本信息（车牌、所属公司、运营状态）
- 实时数据（速度、电池电量）
- 运营信息（运营路线、最后更新时间）
- 合规状态标识

#### 操作功能

- 轨迹追踪（预留功能）
- 联系运营商（预留功能）
- 分享车辆信息

### 3. 审批管理页

#### 审批流程

- 待审核 → 审核中 → 已通过/已驳回

#### 功能特点

- 分类筛选（全部/待审核/已通过/已驳回）
- 审批操作（通过/驳回）
- 下拉刷新
- 审批状态可视化

### 4. 扫码核验页

#### 核验流程

1. 点击扫描按钮启动相机
2. 扫描车辆二维码
3. 显示核验结果
4. 查看车辆详情

#### 核验内容

- 车牌号
- 运营单位
- 运营状态
- 合规状态
- 驾驶模式

  "id": 1,
  "action": "approve" | "reject"
}

## 数据可视化

### 首页数据卡片

使用卡片式布局展示关键指标：

- 总车辆数（蓝色标识）
- 在线车辆（绿色标识）
- 异常车辆（红色标识）
- 今日新增（橙色标识）

### 地图展示

- 使用微信小程序 `map` 组件
- 自定义标记图标（根据状态显示不同颜色）
- 点击标记显示车辆信息气泡
- 支持缩放和拖动

### 图表组件

自定义 Canvas 图表组件，支持：

- 折线图（车辆运行趋势）
- 柱状图（报警趋势）
- 自动适配尺寸
- 触摸交互

## 使用说明

### 安装依赖

本项目为纯原生小程序，无需额外安装依赖。

### 运行项目

1. 打开微信开发者工具
2. 导入项目目录
3. 填入 AppID（或使用测试号）
4. 点击编译运行

### 功能测试

#### 首页功能

1. 启动后自动加载统计数据
2. 地图显示车辆位置
3. 图表展示趋势数据
4. 点击地图标记查看车辆详情

#### 审批功能

1. 进入审批页面
2. 查看待审批列表
3. 点击通过/驳回按钮
4. 确认操作结果

#### 扫码功能

1. 进入核验页面
2. 点击扫描按钮
3. 扫描车辆二维码
4. 查看核验结果

## Mock 数据

项目使用本地 Mock 数据，无需后端支持。所有接口数据定义在 `utils/mock.js` 文件中。

### 数据说明

- 车辆列表：8 辆无人车，分布在上海市
- 审批列表：3 条审批记录（待审核/已通过/已驳回）
- 图表数据：最近 7 天的趋势数据

## 扩展功能

### 可添加的功能

1. **实时数据推送**：使用 WebSocket 实现实时数据更新
2. **历史轨迹回放**：展示车辆历史行驶轨迹
3. **报警消息推送**：实时推送车辆异常报警
4. **数据导出功能**：导出统计数据为 Excel
5. **权限管理**：不同角色查看不同数据
6. **地图搜索**：搜索特定车辆或区域

### 后端对接

如需对接真实后端：

1. 修改 `utils/request.js` 中的接口地址
2. 替换 `utils/mock.js` 为真实接口调用
3. 添加数据校验和错误处理

## 注意事项

1. **地图使用**：需在微信公众平台配置地图服务
2. **扫码权限**：需在 app.json 中声明扫码权限
3. **tabBar 图标**：已生成占位图标，可替换为正式图标
4. **调试模式**：开发时可使用模拟数据进行调试
