## 🚀 如何运行

### 1. 打开微信开发者工具

下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

### 2. 导入项目

1. 打开微信开发者工具
2. 点击"导入项目"
3. 选择项目目录：`/Users/lkpkp/WeChatProjects/miniprogram-2`
4. 填入 AppID（可使用测试号）
5. 点击"确定"

### 3. 编译运行

1. 点击工具栏的"编译"按钮
2. 等待编译完成
3. 在模拟器中查看效果

## 📱 功能演示

### 首页（数据大屏）

打开后自动显示：

- **顶部**：平台标题和副标题
- **统计数据**：4 个卡片展示关键指标
- **地图区域**：显示 8 辆无人车位置
- **图表区域**：
  - 左：车辆运行趋势（折线图）
  - 右：报警趋势（柱状图）

**交互**：
- 点击地图上的车辆标记 → 跳转到车辆详情页
- 下拉刷新数据

### 车辆详情页

从首页点击车辆标记进入：

- **头部**：车辆图标、名称、车牌、状态标签
- **基本信息**：所属公司、运营状态、合规状态、驾驶模式
- **实时数据**：速度、电池电量（卡片形式）
- **运营信息**：运营路线、最后更新时间
- **操作按钮**：轨迹追踪、联系运营商

### 审批管理页

点击底部导航"审批"进入：

- **顶部**：页面标题
- **Tab 栏**：全部 / 待审核 / 已通过 / 已驳回
- **审批列表**：显示所有审批记录
- **操作按钮**：驳回（红色）、通过（绿色）

**交互**：
- 点击 Tab 切换筛选
- 点击"通过"按钮 → 确认弹窗 → 更新状态
- 点击"驳回"按钮 → 确认弹窗 → 更新状态
- 下拉刷新列表

### 扫码核验页

点击底部导航"核验"进入：

- **初始状态**：
  - 扫描图标（带动画）
  - "点击扫描车辆二维码" 提示
  - 使用说明
  
- **扫描成功**：
  - 核验结果卡片（通过/未通过）
  - 车辆详细信息
  - "继续扫描" / "查看详情" 按钮

**交互**：
- 点击扫描图标 → 启动相机扫码
- 扫描成功 → 显示核验结果
- 点击"查看详情" → 跳转到车辆详情页

## 🔧 扩展开发

### 添加真实后端接口

1. 修改 `utils/request.js` 中的 `API_BASE_URL`
2. 替换 `utils/mock.js` 中的模拟数据为真实接口调用
3. 添加错误处理和加载状态

### 添加新功能

1. 在 `pages/` 下创建新页面目录
2. 在 `app.json` 中注册新页面路由
3. 添加对应的 `.js`、`.wxml`、`.wxss`、`.json` 文件

### 优化图标

当前使用占位图标，可替换为正式图标：

1. 准备 PNG 格式图标（60x60 像素）
2. 替换 `assets/images/` 目录下的对应文件
3. 保持文件名不变
